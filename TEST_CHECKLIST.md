# 搜尋功能測試清單

## 功能測試

### 前端搜尋欄測試
- [ ] 搜尋欄顯示在頁面上（書籍標題下方）
- [ ] 搜尋欄包含輸入框、搜尋按鈕和清除按鈕
- [ ] 搜尋按鈕有放大鏡圖標
- [ ] 清除按鈕有 X 圖標
- [ ] 搜尋欄在響應式設計中正常顯示

### 搜尋功能測試
1. **搜尋書籍標題**
   - [ ] 在搜尋框中輸入 "Python" 並點擊搜尋
   - [ ] 應該只顯示標題包含 "Python" 的書籍
   - [ ] 結果數量應該更新

2. **搜尋作者名稱**
   - [ ] 在搜尋框中輸入作者名稱並點擊搜尋
   - [ ] 應該只顯示該作者的書籍
   - [ ] 搜尋應該是模糊的（例如 "Knut" 應該找到 "Knuth"）

3. **Enter 鍵快速搜尋**
   - [ ] 在搜尋框中輸入文字
   - [ ] 按下 Enter 鍵
   - [ ] 應該直接執行搜尋（無需點擊按鈕）

4. **清除搜尋**
   - [ ] 執行搜尋後，點擊清除按鈕
   - [ ] 搜尋框應該被清空
   - [ ] 應該加載所有書籍

5. **分頁與搜尋**
   - [ ] 搜尋時分頁應該重置到第一頁
   - [ ] 搜尋結果應該支持分頁
   - [ ] 前進/後退按鈕應該在搜尋結果中工作

6. **視圖切換**
   - [ ] 搜尋結果應該在網格和列表視圖中都正常顯示
   - [ ] 視圖切換不應該清除搜尋條件

### 邊界情況測試
- [ ] 搜尋空字符串應該顯示所有書籍
- [ ] 搜尋不存在的內容應該顯示 "No books found" 消息
- [ ] 搜尋大小寫不敏感（例如 "python" 和 "PYTHON" 應該返回相同結果）
- [ ] 特殊字符搜尋（例如單引號、引號等）應該被正確處理

### API 測試
使用瀏覽器的開發者工具或 API 測試工具：

1. **GET /api/books?search=PHP&page=1&perPage=10**
   - [ ] 應該返回搜尋結果
   - [ ] 結果應該包含書籍標題或作者名稱中包含 "PHP" 的書籍

2. **GET /api/books?title=Python&page=1&perPage=10**
   - [ ] 應該返回標題中包含 "Python" 的書籍

3. **GET /api/books?authorName=Author&page=1&perPage=10**
   - [ ] 應該返回作者名稱中包含 "Author" 的書籍

4. **GET /api/books?page=1&perPage=10**
   - [ ] 無搜尋參數應該返回所有書籍
